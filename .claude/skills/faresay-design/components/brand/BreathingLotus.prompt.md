Faresay's signature hero motif — a lotus whose petals slowly close and open on a 4-7-8 breathing cadence. Pure CSS, respects `prefers-reduced-motion`.

```jsx
<BreathingLotus size={240} />
```

Use it as the centrepiece of calm hero sections (the therapist-home hero). Self-contained — injects its own keyframes, so it works in any page. Don't surround it with other motion; it should be the only thing moving.
