Rounded-square avatar tile (rounded-2xl) with a brand-tinted background and Fraunces initials — matches the therapist cards.

```jsx
<Avatar name="Priya Patel" />
<Avatar name="Tom Reed" src="/photo.jpg" size={64} />
```

Faresay avatars are soft squares, **not** circles. Initials are auto-derived from `name`; pass `src` for a real photo.
