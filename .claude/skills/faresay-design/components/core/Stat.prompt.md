Headline stat tile — the "You pay £40–55 / Therapist keeps 90% / Our fee 15%" cards from the homepage and dashboards.

```jsx
<Stat label="You pay" value="£40–55" sub="per session" />
<Stat label="Our fee" value="15%" sub="fixed, transparent" accent="coral" />
```

Big Fraunces value over a muted label and sub. `accent` colours the number — `brand` (default), `coral`, or `dark`. Group three across in a grid for the classic comparison band.
