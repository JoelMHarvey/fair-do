Quiet sand-tinted chip for specialisms and approaches — the small tags clustered on therapist cards.

```jsx
<div style={{ display:'flex', gap:6, flexWrap:'wrap' }}>
  <Tag>Anxiety</Tag><Tag>CBT</Tag><Tag>EMDR</Tag>
</div>
```

Lighter than a `Badge` — use `Tag` for sets of attributes, `Badge` for status/credentials.
