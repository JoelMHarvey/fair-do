Soft pill label for credentials, "Founding" status, ratings and feature flags — quiet, tinted, never loud.

```jsx
<Badge tone="coral" icon="★">Founding</Badge>
<Badge tone="amber" icon="★">4.9 (32)</Badge>
<Badge tone="brand" icon="✓">BACP verified</Badge>
```

Tones: `neutral`, `brand`, `solid`, `coral` (founding/highlight), `amber` (ratings only), `outline`. Pass `icon` for a leading glyph — Faresay uses ★, ✓ and the occasional emoji.
