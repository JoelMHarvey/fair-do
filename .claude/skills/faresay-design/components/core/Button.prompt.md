Pill-shaped action button — the primary CTA across Faresay; calm, with a brand-tinted shadow and a gentle 1px hover lift.

```jsx
<Button href="/sign-up">Start free</Button>
<Button variant="secondary">See how it works</Button>
<Button variant="ghost" size="sm">Learn more</Button>
```

Variants: `primary` (brand-600 fill, default), `secondary` (white + sand border), `ghost` (soft brand outline), `inverse` (white on dark sections), `coral` (warm accent). Sizes: `sm` · `md` · `lg`. Pass `href` to render an `<a>`, or `fullWidth`. Buttons are always fully rounded — never square corners.
