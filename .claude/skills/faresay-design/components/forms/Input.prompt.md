Labelled text field with a soft brand focus ring (border → brand-600, 3px halo). White surface, sand border, calm.

```jsx
<Input label="Your name" placeholder="Jane Doe" />
<Input label="Bio" multiline rows={5} hint="A few warm sentences about your practice." />
```

Set `multiline` for a textarea. `label` and `hint` are optional. Inputs use 12px (radius-md) corners — softer than square, not full pills.
