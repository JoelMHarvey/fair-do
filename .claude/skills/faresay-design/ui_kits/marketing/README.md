# Marketing site UI kit

A faithful recreation of the **current** Faresay homepage — the therapist-first practice portal (`src/components/TherapistHome.tsx`). Faresay is a simple monthly **subscription** tool, not a per-session fee.

- **index.html** — the full homepage, top to bottom.
- **pricing.html** — the `/pricing` page: Starter free · Practice £29/mo · Clinic tiers, founding-pricing badge, and an interactive FAQ accordion.
- **Chrome.jsx** — sticky `Nav` (blurred sand bar, Features / Pricing / Find a therapist) + `Footer` (4-column).
- **Sections.jsx** — `Hero` (with the breathing lotus), `EthosBand` ("Made by people with heart"), `Benefits` (6-card grid), `HowItWorks` ("Up and running this week"), `PrivacyBand`, `FinalCTA`.
- **Pricing.jsx** — the tier cards + FAQ accordion (`Pricing`).

Built on the design-system bundle (`Button`, `Card`, `BreathingLotus`, `Logo`). Copy is lifted verbatim from the product.

> The earlier 15%-fee marketplace homepage ("Therapy that works. Priced fairly.") is the **previous** model — the product has since pivoted to this therapist-first practice portal with subscription pricing (Starter free · Practice £29/mo).
